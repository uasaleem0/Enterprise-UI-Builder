This project follows the **GitHub Flow** workflow:

1. Create branch: `git checkout -b feat/your-feature`
2. Make changes and commit
3. Push branch: `git push -u origin feat/your-feature`
4. Open Pull Request
5. Wait for CI/CD to pass
6. Review and merge
